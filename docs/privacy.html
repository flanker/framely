Privacy Policy

Last updated: April 30, 2026

Framely ("the extension") is a Chrome extension that captures the currently visible browser tab and wraps the screenshot in a stylish browser frame so you can share polished mockups. This page explains what data Framely handles and how.

Summary

  • Framely runs entirely on your device.
  • No personal data, browsing history, or screenshots are sent to any remote server.
  • No analytics, tracking, advertising, or third-party SDKs are included.
  • No accounts, sign-in, or payments.

What Framely accesses

  • The visible area of the active tab — only when you click the Framely toolbar icon. The pixels are captured via Chrome's tabs.captureVisibleTab API and used to render the framed mockup.
  • Local browser storage — used as a temporary handoff (key framely-screenshot) when the active page cannot host the in-page overlay (for example chrome:// pages) and Framely opens a viewer tab instead. This entry is overwritten on the next capture.
  • Host access on all sites — required so that the screenshot capture and overlay can work on any page you choose to invoke Framely from. Framely does not read page contents beyond rendering its own overlay UI.

Data sharing

Framely does not transmit, sell, share, or otherwise disclose any data to third parties. Captured screenshots stay on your device unless you explicitly export or share them yourself.

Remote code

Framely does not load or execute any remote code. All scripts are bundled with the extension at publish time.

Children's privacy

Framely is not directed at children under 13 and does not knowingly collect personal information from anyone.

Changes to this policy

If this policy changes, the updated version will be published at this same URL with a new "Last updated" date.
